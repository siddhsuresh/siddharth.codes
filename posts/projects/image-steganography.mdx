# CloakShare - End to End Encryption Native Application built using Tauri

This project for the course Cryptography and Network Security, created an end to
end secure file transfer application secured by RSA and AES algorithms. The app
performs Image Steganography using the DCT method. Comparative analysis with
other methods and easy to use desktop application with focus on UI/UX.
